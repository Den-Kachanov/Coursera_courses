# Python GUI Development with PyQt — Coursera

Цей репозиторій містить увесь код, написаний під час проходження спеціалізації **Python GUI Development with PyQt** на Coursera (рівні Beginner та Intermediate). Курс побудований на базі PyQt5, однак увесь код у цьому репозиторії написано з використанням **PyQt6** — новішої та актуальнішої версії фреймворку. Більшість API між двома версіями є ідентичним, відмінності стосуються лише просторів імен під час імпорту та роботи з флагами.

## Самостійний проєкт — Веб-браузер на PyQt6

Як підсумковий проєкт було розроблено повноцінний десктопний **веб-браузер** з використанням PyQt6 та `PyQt6-WebEngine`. Застосунок побудований на трьох власних класах і має повністю кастомізований темний інтерфейс, описаний через QSS-таблицю стилів та `QPalette`.

Вихідний код знаходиться в директорії [`project/`](./project).

### Архітектура

Проєкт складається з трьох класів:

**`BrowserTab(QWebEngineView)`** — Підклас `QWebEngineView`, що представляє одну вкладку браузера. Перевизначає `createWindow()` для перехоплення запитів на відкриття нових вікон (наприклад, посилання з `target="_blank"`) і перенаправляє їх у нові вкладки всередині того самого застосунку замість створення окремого вікна.

**`ListDialog(QDialog)`** — Перевикористовуваний модальний діалог побудований на `QListWidget`. Приймає список пар `(назва, url)` і відображає їх як вибірні елементи. Відправляє власний сигнал `pyqtSignal` (`url_selected`) по подвійному кліку, який підключається зовні до логіки навігації. Використовується як для панелі закладок, так і для панелі історії.

**`BrowserWindow(QMainWindow)`** — Головне вікно застосунку. Керує побудовою інтерфейсу, життєвим циклом вкладок, станом навігації та взаємодією з користувачем.

### Функціональність

**Вкладки.** Реалізовані через `QTabWidget` у режимі документа з підтримкою закриття та перетягування вкладок. Назви вкладок обрізаються до 25 символів з додаванням "…". Кнопка "+" для відкриття нових вкладок розміщена у правому куті панелі вкладок через `setCornerWidget()`.

**Панель навігації.** Містить кнопки "Назад", "Вперед", "Оновити/Зупинити" та "Додому", побудовані через спільний фабричний метод `_nav_btn()`. Кнопка оновлення динамічно змінюється на "Зупинити" ("✕") під час завантаження сторінки шляхом відключення та підключення сигналу `clicked`, і повертається до "↻" після завершення.

**Розумний адресний рядок.** Метод `_navigate()` застосовує трьохгілкову логіку обробки введення: якщо рядок починається з `http://`, `https://`, `file://` або `about:` — використовується як є; якщо містить крапку і не має пробілів — автоматично додається `https://`; в іншому випадку запит передається в Google-пошук.

**Індикатор безпеки.** Значок `QLabel` ліворуч від адресного рядка оновлюється при кожній зміні URL: зелений значок замку для HTTPS-з'єднань та жовтий попереджувальний значок для звичайного HTTP.

**Індикатор завантаження.** Тонкий `QProgressBar` заввишки 3px вбудовано у другий `QToolBar` під основною панеллю. За замовчуванням прихований, відображається при `loadStarted`, оновлюється при `loadProgress` і ховається через 500мс після `loadFinished` за допомогою `QTimer.singleShot()`.

**Закладки.** Кнопка закладок (☆/★) на панелі інструментів викликає `_toggle_bookmark()`, яка додає або видаляє поточну сторінку зі списку пар `(назва, url)` у пам'яті. Іконка кнопки оновлюється відповідно до стану. Закладки відображаються через `ListDialog`.

**Історія перегляду.** Кожна зміна URL додає запис `(назва, url)` до списку в пам'яті з обмеженням у 200 записів. Дублікати URL видаляються перед вставкою для уникнення повторів. Історія відображається через `ListDialog`.

**Контекстне меню.** `QMenu`, прикріплений до кнопки "⋮" на панелі інструментів, надає доступ до закладок, історії, пошуку на сторінці, оновлення та виходу.

**Клавіатурні скорочення.** Зареєстровані через `QShortcut`:

| Скорочення | Дія |
|------------|-----|
| `Ctrl+T` | Нова вкладка |
| `Ctrl+W` | Закрити поточну вкладку |
| `Ctrl+L` | Фокус на адресний рядок |
| `F5` | Оновити сторінку |
| `Alt+←` | Назад |
| `Alt+→` | Вперед |
| `Alt+Home` | Домашня сторінка |
| `Ctrl+D` | Додати/видалити закладку |
| `Ctrl+H` | Показати історію |
| `Ctrl+B` | Показати закладки |
| `Ctrl+F` | Пошук на сторінці |

**Кастомна домашня сторінка.** Сторінка `about:home` рендериться з вбудованого HTML-рядка (`HOME_HTML`) через `QWebEngineView.setHtml()`. Містить темний фон з трьома анімованими розмитими градієнтними кулями, живий цифровий годинник та дату українською мовою (оновлюються кожну секунду через JavaScript `setInterval`), градієнтний логотип та сітку з восьми ярликів на популярні сайти (Google, YouTube, GitHub, Wikipedia, Twitter, Reddit, Google Translate, Погода).

**Оформлення.** Палітра з 16 кольорів визначена у словнику `COLORS` та підставляється у повну QSS-таблицю стилів, що охоплює всі типи віджетів: `QMainWindow`, `QTabBar`, `QToolBar`, `QPushButton`, `QLineEdit`, `QProgressBar`, `QStatusBar`, `QMenu`, `QDialog`, `QListWidget` та `QScrollBar`. Програмний `QPalette` також налаштований для відповідності темній темі на всіх системних елементах.

**Підтримка HiDPI.** Змінна середовища `QT_ENABLE_HIGHDPI_SCALING` встановлюється перед запуском застосунку для коректного масштабування на екранах з високою роздільною здатністю.
